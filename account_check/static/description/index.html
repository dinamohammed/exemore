Bank checks

New features, new stages

Usage

This module add full cycle of checks.
- Translation of all the terms involved.
- Simplification of management in general.

Journal Payment Methods modifications:-

Add new Journal Payment Method types.
- Received Checks for customer received payments as checks.
- Delivered third checks as checks received but will be use in issue operation later.
- Issued checks to use for pay vendor payments.

Payment modifications:-

Add new fields for payment methods of journals with new payment methods for checks.

Create Check:-
